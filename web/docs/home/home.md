Installing Web-Standards-App
===

Vooraf:
- Mamp in orde brengen
- Gulpfile browserSync proxy juist zetten
<br />

Requirements
---

* [node.js]
* [composer]
* [bower]
* [bundler]
* [gulp]
<br />

Terminal commands
---

    $ npm install

    $ composer install

    $ bower install

    $ bundle install

    $ gulp build

    $ gulp build--styleguide
